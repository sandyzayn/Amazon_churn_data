# 🚀 Customer Churn Prediction using Logistic Regression

## 📌 Project Overview

This project focuses on predicting whether a customer will **churn (leave)** or **stay** using Machine Learning.
We use **Logistic Regression**, a classification algorithm, to analyze customer behavior and make predictions.

---

## 🎯 Objective

* Identify customers likely to churn
* Help businesses take preventive actions
* Improve customer retention using data-driven insights

---

## 📊 Dataset Description

The dataset contains customer-related features:

| Feature      | Description                    |
| ------------ | ------------------------------ |
| CustomerID   | Unique ID of customer          |
| Age          | Customer age                   |
| MonthlySpend | Monthly spending               |
| Tenure       | Duration with company (months) |
| NumOrders    | Number of orders placed        |
| Churn        | 0 = Not Churn, 1 = Churn       |

## 🧠 Model Used

---

## 📏 Evaluation Metrics

---

## 📊 Results

* Achieved good performance in predicting churn customers
* Balanced evaluation using **F1 Score** for better insights

---

## 📉 Visualization

* Bar chart showing churn vs non-churn distribution

---

## 💡 Key Insights

* Customers with **low tenure** are more likely to churn
* Lower spending and fewer orders indicate higher churn risk
* Behavioral data plays a key role in prediction
